Your Evolution API instances

Each name below is a separate WhatsApp connection. Create matching instances in Evolution API, then scan the QR code for each one. By default the primary instance name is your Account ID (same UUID, lowercase, with hyphens). A 32-character hex name without hyphens is treated as the same account for routing.

Two or more WhatsApp numbers

Your admin adds extra names as a comma-separated list in your account (or appends one at a time). Examples: 4724b106-8497-4191-88a0-1b3f09722ee2 (primary) plus 4724b1068497419188a01b3f09722ee2_sales — each segment must be created separately in Evolution Manager and linked with its own QR scan. The campaign dropdown lists all names; choose Auto on the dashboard to rotate across every line you own.

Admin: POST /admin/users/<id>/evolution-instances with body {"instances":"name1,name2,name3"}, or POST .../evolution-instances/append with {"instance":"extra_name"} to add without replacing the list.
